print(sys.argv[2]) # "123"

來使用這些參數。

任務

1. 閱讀 convert.py

請閱讀 convert.py，了解這份 python 程式是如何運作的。

2. 加入 Error Handling

在假設 .read()、.write() 系列的程式不會出錯的情況下，幫 convert.py 加上 Error Handling 的程式。


你的 convert.py 必須至少處理以下幾種類型的錯誤：

a. 檔案相關錯誤

  • input 檔案不存在
  • input 路徑是資料夾
  • output 路徑是資料夾
  • output 的資料夾不存在

處理方式：

  • 程式必須直接結束，使用 sys.exit(52)
  • exit code 必須為 52

b. deadline 錯誤

  • 格式錯誤
  • 無法被 datetime 正確解析

處理方式：

  • 程式必須直接結束，使用 sys.exit(52)
  • exit code 必須為 52

c. 欄位為空

  • name 為空
  • score 為空
  • timestamp 為空

處理方式 (優先級由前往後)：

  • name 為空 -> 跳過這一筆紀錄
  • score 為空 -> 表格中的所有成績欄位變為X
  • timestamp 為空 -> 表格中的最終成績欄位變為X

d. score 錯誤

  • 不是數字 (例如 abc)
  • 超出範圍 (小於 0 或大於 100)

處理方式：

  • 表格中的所有成績欄位變為X

e. timestamp 錯誤

  • 格式錯誤
  • 無法被 datetime 正確解析

處理方式：

  • 表格中的最終成績欄位變為X

f. 整行格式錯誤

  • 無法被正確解析成 name / score / timestamp

處理方式：

  • 跳過這一筆紀錄

執行

在 CLI 執行：

python convert.py input.txt output.txt "2026-04-04 12:00"

參數說明：

  • input.txt -> 輸入日誌檔案
  • output.txt -> 輸出表格檔案
  • deadline -> 截止時間
